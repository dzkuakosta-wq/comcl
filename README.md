## Community Club Landing (HTML/CSS/JS)

Базовая структура лендинга community‑клуба: описание клуба, форма регистрации, пример профиля участника и поиск по интересам.

### Структура проекта

- `index.html`: главная (о клубе, преимущества, регистрация, быстрый поиск участников)
- `profile.html`: пример профиля участника (просмотр/редактирование)
- `search.html`: отдельная страница поиска по интересам
- `assets/css/styles.css`: стили
- `assets/js/data.js`: демо‑данные и утилиты для работы с `localStorage`
- `assets/js/app.js`: логика страниц (регистрация, поиск, профиль)

### Запуск

Откройте `index.html` в браузере.

### Как это работает

- Регистрация на главной странице сохраняет участника в `localStorage` и открывает его профиль.
- Поиск по интересам фильтрует список участников (демо‑данные + зарегистрированные).
- Профиль можно отредактировать — изменения сохраняются в `localStorage`.
